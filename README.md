# E-Commerce SQL Analytics Project

## Project Overview
This project uses MySQL and Python to show an end-to-end data analytics workflow.
The objective was to extract meaningful business insights about revenue, customer behavior,
engagement, and churn by cleaning raw e-commerce data, loading it into a relational database, and doing SQL-based analysis.


## Tech Stack
- MySQL
- Python (Pandas)
- MySQL Workbench

## Dataset
Four e-commerce datasets were used:
- Customers
- Products
- Orders
- Order Items

## Data Cleaning & ETL
- Cleaned raw CSV data using Python (Pandas)
- Removed duplicates, handled missing values, and validated data types
- Loaded cleaned data into MySQL using a Python ETL script

## SQL Analysis
- Developed 15+ SQL queries using joins, aggregations, CTEs, and window functions
- Analyzed revenue, customer behavior, MAU, and churn

## Key Insights
- Total revenue: ₹2.19L
- Electronics contributed 96% of sales
- Stable Monthly Active Users over 5 months
- Customer churn: 0%
